Variaves Html

Int

Int: Variavel do tipo inteiro, positivo ou negativo. Exemplo: O numero de bolas em uma caixa

Float

Float: Variável numérica do tipo decimal. Exemplo: O comprimento de um sapato

Double

Double: Variável numérica do tipo decimal de precisão dupla. Exemplo: Juros de uma compra

Char

Char: Variável que representa um caractere do tipo texto. Exemplo: O nome de uma pessoa

String

String: Variável que representa um conjunto de caracteres do tipo texto. Exemplo: Uma frase

Boolean

Boolean: Variável booleana, ou seja, que pode representar somente dois valores, verdadeiro ou falso. Exemplo: A correção de uma atididade feita por um aluno
